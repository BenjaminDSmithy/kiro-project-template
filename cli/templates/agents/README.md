# Repository Skills

This directory contains repository-scoped Codex skills that can be shared with a team.

Place reusable skills under `.agents/skills/<skill-name>/SKILL.md`.

Suggested usage:

- keep each skill focused on one job
- prefer instruction-only skills unless you need scripts
- mention a skill explicitly with `$skill-name` when you want deterministic routing

The generated starter currently includes:

- `release-check`
- `qa-regression`
- `docs-audit`
